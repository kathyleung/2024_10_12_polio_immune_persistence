The codes in the folder "sensitivity_analysis_gender" are for conducting sensitivity analysis on gender differences.

Please run the codes following the below order:

1) main.m: This file contains the code for performing statistical inference in a Bayesian framework with noninformative (flat) priors using Markov Chain Monte Carlo with Gibbs sampling by gender. 

2) sex_diff_mean_titre.m: This file contains the code for estimating the difference in mean titre distributions between the two genders over time.
