The codes in the folder "sensitivity_analysis_reduced_waning" are for sensitivity analysis on waning rate and generating Figure S1.

Please run the codes following the below order:

1) main_polio_sensitivity_Ma.m: This file contains the code for performing statistical inference in a Bayesian framework with noninformative (flat) priors using Markov Chain Monte Carlo with Gibbs sampling assuming that the waning rates of the mean nAb titres declined between 18 months and 10 years at the same rates in Ma et al.
