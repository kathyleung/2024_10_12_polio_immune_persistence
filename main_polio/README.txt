The codes in the folder "main_polio" are for generating Figure 1.

Please run the codes following the below order:

1) main_polio.R: This file contains the code for performing statistical inference in a Bayesian framework with noninformative (flat) priors using Markov Chain Monte Carlo with Gibbs sampling. 

2) main_polio_plot.m: This file contains the code for estimating the distribution of mean titre and seroprotection proportion over time, and generating Figure 1.
